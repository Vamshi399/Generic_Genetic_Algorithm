crossover(parents, population, percent) → {Array.<Element>}

Given a population it generates a set of children formed after mating of his parents. Mating is to take the first half of the first parent chromosome and the second half of the second father, to concatenate and form the child element. A percentage of children who will be the result of crosses is set, the rest will be the best elements of the population.
Parameters:
Name Type Description
parents Array.<Element> Elements chosen to reproduce after a selection function.
population Array.<Element> A set of Element type elements that are now the population.
percent number Percentage of children who will be the calculated as a result of crosses.
