crossover(parents, population, percent) → {Array.<Element>}

Dada una poblacion genera un conjunto de hijos formados tras el apareamiento de sus padres. El apareamiento consiste en coger la primera mitad del cromosoma del primer padre y la segunda mitad del segundo padre para concatenarlos y formar el elemento hijo. Se fija un porcentaje de hijos que serán el resultado de cruces, el resto serán los mejores elementos de la poblacion.
Parameters:
Name Type Description
parents Array.<Element> Elementos elegidos para reproducirse tras una funcion de seleccion.
population Array.<Element> Conjunto de elementos de tipo Element que forman actualmente la poblacion.
percent number Porcentaje de los hijos calculados que serán resultado de cruces.
