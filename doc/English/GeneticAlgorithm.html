simulateGeneration()

Simula el proceso que vive una generacion de elementos del algoritmo genetico. - Se inicializa una poblacion - Se evaluan sus elementos con una función de fitness. - Una vez obtenidos los valores de fitness se procesa una función de selección para escoger que elementos se reproducirán. - Se lleva a cabo el proceso de reproducción. - Se lleva a cabo el proceso de mutación. - Se calcula el mejor elemento de la población - El numero de generación avanza
